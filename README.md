# Flask To-Do App

A simple to-do list app built using Flask.

## Features
- Add tasks
- Delete tasks
- Update tasks

## Run locally
pip install flask  
python app.py

## 📌 Description

A simple To-Do List web application built using Flask.
Users can add, update, and delete tasks.

## 🛠️ Technologies Used

* Python (Flask)
* HTML
* CSS
* JavaScript
* SQLite

## ▶️ How to Run

1. Install dependencies:
   pip install flask

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/
